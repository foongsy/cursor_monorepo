# Plan with linear

## Overview
Given a specific Linear issue to solve, generate a plan and update the issue accordingly.

1. **Understand the issue**
   - Use Linear MCP to retrieve information about the issue
   - If the issue is completed, do not continue
   - If the issue is not yet done by anyone, understand what the issue is about
   - Clarify with the user if you have any confusion

2. **Update the issue with a plan**
   - Generate a plan with the understanding and clarification
   - Update the issue with the plan details
   - Add screenshots if applicable
   - Create sub-issue according to the TO DO items
